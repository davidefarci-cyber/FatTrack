# BATCH 05 / 10 — illustrazioni esercizi FatTrack

## Istruzioni per la chat GPT

Genera UNA immagine per ognuno dei 7 prompt elencati sotto. Ogni prompt è auto-contenuto: stile, palette e formato sono ripetuti in ogni blocco. Segui ESATTAMENTE il prompt di ciascuno.

Al termine di tutte le 7 immagini, raggruppale in un singolo file ZIP chiamato `fattrack-exercises-batch-05.zip` con i PNG nominati esattamente come indicato nel campo "**Nome file**" di ogni blocco. I nomi devono essere:

- high-knees.png
- skater-jumps.png
- squat-jumps.png
- cat-cow.png
- cobra.png
- child-pose.png
- downward-dog.png

Niente cartelle interne nello ZIP, file PNG flat alla radice.

## Regole comuni a tutte le immagini di questo batch

- Formato PNG, dimensioni indicate nel singolo prompt.
- Sfondo bianco puro (#FFFFFF) o trasparente.
- Palette rigida: #FF7A1A / #D45C00 / #FFE0C8 / #1E2532 / #FFFFFF. Nessun altro colore.
- NESSUN testo nelle immagini, NESSUNO sfondo scenografico, NESSUNA freccia/etichetta/numero.
- Stile vettoriale flat moderno, coerente tra tutte le immagini.
